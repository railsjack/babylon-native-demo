Class AnimationPlayer

Animation Player

param sceneManager

Object containing Camera, Scene, reset Function and render function

param animations

Array containing the list of animations

param meshes

Array of all the mesh of the 3D model

param dict

Array containing the correspondence between mesh.id and mesh.userData.myId

Array containing the initial state of all mesh (position, colors, rotation, visibility, scale, refItem)
